# Quiz Application (SIT708 Task 3.1)

## Overview
This project is an Android Quiz Application developed as part of SIT708.  
The application allows users to enter their name, attempt a multiple-choice quiz, track their progress, and view their final score.

---

## Features

- User enters name before starting quiz  
- Multiple-choice questions  
- Answer feedback:
  - Correct answer shown in green  
  - Wrong answer shown in red  
- User cannot change answer after submission  
- Progress bar updates during quiz  
- Final result screen shows score  
- "Take New Quiz" keeps user name  
- "Finish" closes the app  
- Dark Mode / Light Mode toggle available on all screens  

---

## Technologies Used

- Android Studio  
- Kotlin  
- XML Layouts  
- SharedPreferences (for storing name and theme)  

---

## App Flow

1. User enters name on main screen  
2. User starts quiz  
3. Questions are displayed one by one  
4. User selects answer and clicks submit  
5. Feedback is shown  
6. User moves to next question  
7. After last question, result screen is displayed  

---

## How to Run

1. Open project in Android Studio  
2. Connect Android device or use emulator  
3. Click Run  
4. Install and launch app
